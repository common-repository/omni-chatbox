== Description ==

### Omni Chatbox

Omni Chatbox is a platform that allows you to quickly integrate __AI chatbots__ into your website.

#### Chatbots engines:

* AWS Connect

It supports all features from the AWS Connect.

> Note: You can use Omni Chatbox for Free. Paid subscription if you need additional capabilities.

For more details, visit [omnilegion](http://chatbox.omnilegion.com) and talk to us!


== Installation ==
These instructions assume you have already signed up for Omnilegion and have gone through the setup here [OmniChatbox](http://chatbox.omnilegion.com)

1. Install the "Omni Chatbox" plugin via 'Plugins' menu in Wordpress
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your Chatbox chat ID through the 'Omni Chatbox' menu in WordPress.

== Frequently Asked Questions ==
 
= Does this plugin work with the newest WP version and also older versions? =
Yes, this plugin works fine with the latest version of Wordpress!
It also works great with the old version of WordPress.

= Where can I get the chatbox Id? =
Just register on [Omni Chatbox](http://chatbox.omnilegion.com) and check the publish page. You can find the chatbox id easily.

It is free for 15 days. If you think it is useful for your business, you can upgrade the membership.
